An interactive web-based educational application designed for kids to practice addition up to 20.

## Features

- **Interactive Math Problems**: Random addition questions with numbers that sum to 20 or less
- **Real-time Feedback**: Instant validation with encouraging messages
- **Progress Tracking**: 
  - Score system with bonus points for streaks
  - Success rate percentage
  - Visual progress bar
  - Recent attempts history
- **Statistics Dashboard**:
  - Current score
  - Streak counter
  - Total attempts
  - Correct answers count
- **Persistent Storage**: Progress is saved locally and restored on page reload
- **Kid-Friendly Design**: Colorful gradients, emojis, and engaging animations

## How to Use

1. Open `index.html` in a web browser
2. Read the math question displayed
3. Type your answer in the input field
4. Click "Check Answer" or press Enter
5. See if you're correct and get a new question!
6. Track your progress in the statistics and history sections

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Game logic and progress tracking
